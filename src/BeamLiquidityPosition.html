ABOUT
This app computes the current situation (value, fees, etc.) of a given position in a Liquidity Pool of the Beam DEX. Some indicators are calculated and can be displayed in units of one asset or the other.
The app cannot know if the provided initial position has already been withdrawn (because the Beam blockchain is confidential!), so it assumes it is still currently in place.
This vanilla HTML/JS/CSS code runs locally and only needs to make a few queries to a Beam explorer node: to get the initial position (based on the provided block height or kernel id), to get the asset names and to get the current situation of the pool. All the rest are simple calculations done locally.
